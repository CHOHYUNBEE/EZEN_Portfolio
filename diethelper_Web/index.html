세부 항목 설명서

기초 대사량 산출 방법

- 남자 : 66.47 + (13.75 x 체중) + (5 x 키) - (6.76 x 나이)

- 여자 : 655.1 + (9.56 x 체중) + (1.85 x 키) - (4.68 x 나이)

활동량 선택

- 거의 없음 : 활동이 적거나 운동을 안하는 경우

- 조금 있음 : 가벼운 활동 및 운동을 하는 경우 (1~3일/1주)

- 보통 : 보통의 활동 및 운동을 하는 경우 (3~5일/1주)

- 조금 많음 : 적극적인 활동 및 운동을 하는 경우 (6~7일/1주)

- 정말 많음 : 매우 적극적인 활동 및 운동, 운동 선수

영양 정보 계산법

- 1일 적정량 : 기초 대사량 * 활동량 계수

- 탄수화물량 : 1일 적정량 * 0.5 / 4

- 단백질량 : 1일 적정량 * 0.3 / 4

- 지방량 : 1일 적정량 * 0.2 / 9

식단 타입 선택

- 감량 칼로리 식단 : (적정 칼로리 * 0.8)kcal

- 적정 칼로리 식단 : (적정 칼로리)kcal

- 증량 칼로리 식단 : (적정 칼로리 * 1.2)kcal
