Review

첫 프로젝트

디자인 선정부터 퍼블리싱, GitHub 를 이용한 웹 호스팅 및 유지 보수를 거친 첫 프로젝트로 css와 javascript를 배우는 와중에 중간 중간 제작되어
완성도가 만족스럽지는 않으나 다양한 기술들을 사용해 볼 수 있었고 혼자 이 모든 작업을 하는 게 쉬운 게 아님을 다시 한번 느끼게 되었습니다.

데이터 삽입

아직 DB와 연동하는 부분을 배우지 않았기 때문에 제품 리스트를 불러올 수 있는 방법 을 생각하던 중 js 배열에 데이터를 넣고 불러와서
원하는 카테고리를 뷰포트 크기에 맞는 Grid에 맞춰 구현 해 내는 과정이 새로웠습니다.

아쉬운 점

아직 코드를 작성함에 있어 클래스나 아이디들에 따른 중복된 이벤트들에 대해 정리가 되지 않아 코드 정리가 필요 하다고 생각합니다.

jQuery나 React 등 의 다양한 라이브러리를 사용하여 조금 더 동적인 화면을 구현해 내고 싶습니다.
